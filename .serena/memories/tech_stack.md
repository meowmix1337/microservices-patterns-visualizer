# Tech Stack

## Language
**JavaScript** (ES6+ modules)
- Note: TypeScript migration planned for v3.0

## Framework & Libraries
- **React**: 18.3.1 - UI framework
- **React DOM**: 18.3.1 - DOM rendering
- **Framer Motion**: 11.11.17 - Animation library for smooth transitions and message flows

## Build Tool
- **Vite**: 6.0.3 - Fast build tool and dev server
- **@vitejs/plugin-react**: 4.3.4 - React plugin for Vite

## Dev Dependencies
- **@types/react**: 18.3.12 - TypeScript type definitions for React
- **@types/react-dom**: 18.3.1 - TypeScript type definitions for React DOM

## Package Manager
- **npm** - Version 9.x or higher recommended

## Browser Support
- Chrome/Edge (latest 2 versions)
- Firefox (latest 2 versions)
- Safari (latest 2 versions)
- Mobile browsers (iOS Safari, Chrome Mobile)

## System Requirements
- Node.js 18.x or higher
- macOS (Darwin), Linux, or Windows
- Development currently on Darwin 24.6.0

## Architecture Style
- Component-based React application
- Single Page Application (SPA)
- No backend required (purely frontend visualization)
- Static deployment ready
